# AnchorFlow

AnchorFlow is a small local-first wellbeing and professional-growth check-in app.

- **Anchor** helps choose the one thing that matters and protect space for it.
- **Trail** records the day's terrain, a lesson, and the next small step.
- Entries persist in the browser and can be downloaded or restored as JSON backups.

Run with `npm install` and `npm run dev`. Build with `npm run build`.
